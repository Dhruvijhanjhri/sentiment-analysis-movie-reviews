# Sentiment Analysis of Movie Reviews

This project implements a binary sentiment classification system to classify IMDb movie reviews as positive or negative using Natural Language Processing (NLP) and Machine Learning.

## Tech Stack
- Python
- Pandas, NumPy
- NLTK
- Scikit-learn
- Matplotlib, Seaborn

## Approach
- Text preprocessing: cleaning, tokenization, stop-word removal
- Feature extraction using TF-IDF Vectorization
- Model training using Logistic Regression
- Model evaluation using Accuracy, Precision, Recall, F1-score
- Performance visualization using Confusion Matrix and ROC-AUC Curve

## Results
- Accuracy: 88.76%
- Precision: 87.84%
- Recall: 90.17%
- F1-score: 88.99%

## Dataset
IMDb Movie Reviews Dataset  
https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews
